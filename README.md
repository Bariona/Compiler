## Relevant commands

```
   antlr4 <filename>.g4
   javac *.java
   grun <filename> <rulename> -tokens
   
   diff <file1> <file2> -qZB
```

java编译:

```
javac <filename>.java 
支持中文: javac -encoding UTF-8 <filename>.java
java <filename>
```
`assert` 开区需要IDEA在run configuration中的VM option 加上`-ea`
